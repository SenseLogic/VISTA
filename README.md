# Vista

Modular CSS and JavaScript framework.

## Features

*   Stylus
    *   media queries
    *   class generators
*   JavaScript
    *   server requests
    *   HTML templating
    *   document queries
    *   property animations
    *   parallax backgrounds
    *   carousel strips
    *   reactive components
    *   mathematical types
    *   graphic shaders
    *   model loaders
    *   scene renderers
