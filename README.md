# Vista

Lightweight CSS and JavaScript library.

## Features

*   Stylus
    *   media queries
    *   class generators
*   JavaScript
    *   server requests
    *   HTML templating
    *   document manipulation
    *   property animation
    *   perpetual carousel
    *   parallax effect
