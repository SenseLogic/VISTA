# Vista

Lightweight Stylus and JavaScript framework.

## Features

*   Stylus
    *   media queries
    *   class generators
*   JavaScript
    *   server requests
    *   HTML templating
    *   document queries
    *   property animations
    *   reactive components
    *   graphic shaders
    *   scene graph
